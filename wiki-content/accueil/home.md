## Stack Technique

| Couche | Technologie |
|--------|-------------|
| Backend | Laravel 12 (PHP 8.3+) |
| Frontend | Vue 3 + Inertia.js + Tailwind CSS 4 |
| Admin | Filament 3.x |
| Database | PostgreSQL 16+ / PostGIS |
| Cache | Redis 7+ |
| Search | OpenSearch |
| Storage | Cloudflare R2 + Contabo S3 |
| Payments | Stripe Checkout |
| Real-time | Laravel Reverb |
| Radio | Icecast + AI (OpenRouter + ElevenLabs) |
| Audio Analysis | Python FastAPI + librosa + BirdNET |
